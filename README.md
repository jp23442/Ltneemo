## // filosofia

Não construo pra portfolio. Construo porque o sistema precisa existir.

Cada projeto tem um motivo real:  
Kranio existe porque minha memória não acompanha a quantidade de frentes abertas.  
Neemo existe porque não quero depender de cloud pra pensar.  
Hantavirus existe porque nenhum jogo capturou o que eu quero sentir jogando.  
Forjatec existe porque o chassis que eu queria não existe no mercado.

> *Se você chegou até aqui, provavelmente já entendeu o negócio.*
